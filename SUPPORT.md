# Support

Use GitHub issues for reproducible bugs, documentation problems, and focused
feature requests. Include the YW SkillFoundry version, operating system, Python
and Bash versions, host, command, expected result, actual result, and a minimal
fixture when possible.

Use GitHub Discussions, if enabled, for usage questions and methodology
discussion.

For vulnerabilities, follow `SECURITY.md` and do not disclose them publicly.

Support is best effort. YW SkillFoundry cannot guarantee model behavior,
third-party host compatibility, or the quality of generated skills. The
documented supported execution environment is Python 3.10+, Bash 3.2+, and
macOS/Linux. WSL is expected to work but is not currently CI-verified.
