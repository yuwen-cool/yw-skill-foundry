# 支持

有可复现的 bug、文档问题，或具体功能建议，请开 GitHub Issue。最好写上：

- YW SkillFoundry 版本
- 系统、Python、Bash 版本
- 用的宿主（Codex / Cursor / Claude Code 等）
- 命令、期望结果、实际结果
- 尽量附一份最小可复现材料

使用方式和写法讨论，可以走 GitHub Discussions。

安全问题请按 `SECURITY.md` 私下报告，不要公开细节。

支持是尽最大努力提供，不保证模型行为、第三方宿主兼容性，也不保证生成出来的 skill 一定达标。当前正式支持的运行环境是 Python 3.10+、Bash 3.2+、macOS/Linux；WSL 通常可用，但尚未纳入 CI。
